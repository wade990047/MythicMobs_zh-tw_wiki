用途
=====================

移除指定目標的光環

細項設定
----------

| 技能名稱 | 簡化寫法| 用途 | 預設值 |
|-----------|--------------------------|----------------------|---------------|
| aura  | buff, debuff, name, b, n | 光環名稱 |   |
| stacks| s| 堆疊數量 | Max stacks|

----------

MM v4.12 版更新:

你可以指定名稱 "ANY" 來刪除目標的所有光環

範例
--------
```yaml
  Skills:
  - auraremove{aura=Ice;stacks=10} @self ~onTimer:200
  - ...
```
