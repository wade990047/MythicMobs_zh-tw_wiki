用途
=====================

新增於MM v2.3版本

設置施法者生物的記分板分數

細項設定
----------

| 設定項 | 簡化寫法 | 用途 | 預設值 |
|-----------|---------|----------------------------------------------------------------------------------------------------------------------------------|---------|
| objective | obj, o  | 記分板名稱，若不存在則自動創建 | |
| value | v   | 要使用的數值   | |

  
範例 
----
```yml
  - setmobscore{o=someobjective=;v=2} @self ~onAttack
```
