用途
===================

設定目標實體的移動速度

細項設定
----------

| 預設值 | 簡化寫法 | 用途 | 預設值 |
|------------|-----|-----------------------------------------|---------------|
| speed  | | 速度倍率 | 1 |
| type   | | 加速類型，可以是WALKING, FLYING | WALKING   |

  

範例
--------

```yml
  - setspeed{speed=2;type=walking} ~onSpawn
```
