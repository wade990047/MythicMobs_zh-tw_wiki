用途
===========================

為 `生物` 變更計分板數值

可用的變更動作:

-   SET `設定`
-   ADD `加法`
-   SUBTRACT `減法`
-   MULTIPLY `乘法`
-   DIVIDE `除法`
-   MOD `除後取餘數`

細項設定
----------

| 設定項 | 簡化寫法 | 用途 | 預設值 |
|-----------|---------|----------------------------------------------------------------------------------------------------------------------------------|---------|
| objective | obj, o  | 記分板名稱，若不存在則自動創建 | |
| action| a   | 要進行的動作  | ADD |
| value | v   | 要使用的數值   | |

  
範例
----
```yaml
  - modifymobscore{o=someobjective;a=multiply;v=2} ~onAttack
```
