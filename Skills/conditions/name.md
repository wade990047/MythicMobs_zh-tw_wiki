## Description
Checks against the name of the target entity


## Attributes

| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| name      | n         | The name to check against                                            | Ashijin |


## Examples
```yaml
  Conditions:
  - name{n=Ashijin}
```
