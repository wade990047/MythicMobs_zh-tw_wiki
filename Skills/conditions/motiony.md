## Description
Checks the Y motion of the target entity against a range.


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| velocity  | v         | The velocity to check for                                            |         |


## Examples
```yaml
  Conditions:
  - motiony{v=<1} true
```
