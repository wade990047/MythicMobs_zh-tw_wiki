## Description
Checks if the target is within the given WorldGuard region.


## Attributes

| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| region    | r, name, n| The region to check                                                  |         |


## Examples
```yaml
  Conditions:
  - region{r=someregion} true
```


## Aliases
- [x] inregion
