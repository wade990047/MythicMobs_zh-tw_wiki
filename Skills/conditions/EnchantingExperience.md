## Description
Checks the experience points that the target player has.


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| level     | l         | The experience points to check for. Accepts ranges.                  | 0       |


## Examples
```yaml
  TargetConditions:
  - enchantingExperience{l=>100} true
```
