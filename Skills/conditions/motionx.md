## Description
Checks the X motion of the target entity against a range.


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| velocity | v          | The velocity to check for                                            |         |


## Examples
```yaml
  Conditions:
  - motionx{v=<1} true
```


## Aliases
- [x] motx
