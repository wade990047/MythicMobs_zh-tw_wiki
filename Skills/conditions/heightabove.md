## 用途
Checks if the target's Y location is above a value


## 細項設定

| 技能名稱 | 簡化寫法| 用途 | 預設值 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| height| h | The height they must be above   | |


## 範例
```yaml
  Conditions:
  - heightabove{h=50} true
```
