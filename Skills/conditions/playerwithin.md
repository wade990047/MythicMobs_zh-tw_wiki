## Description
Checks if any players are within a radius of the target.


## Attributes

| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| distance  | d         | The radius to check in                                               | 0       |


## Examples
```yaml
  Conditions:
  - playerwithin{d=10} true
```

- [x] playerswithin
