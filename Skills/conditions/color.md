## Description
Checks the color of Sheeps, Shulkers, Cat (Type), Parrots (Variants), Horses, Llamas, and TraderLlamas.


## Attributes

| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
|   color   | clr, c    | the color to check for                                               | WHITE   |


## Examples
```yaml
  Conditions:
  - color{c=RED} true
```
