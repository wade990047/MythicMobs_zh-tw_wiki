## 用途
Checks if the target's Y location is below a given value


## 細項設定

| 技能名稱 | 簡化寫法| 用途 | 預設值 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| height| h | The height they must be below   | |


## 範例
```yaml
  Conditions:
  - heightbelow{h=50} true
```
