## Description
Checks the food amount of the target


## Attributes

| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| amount    | a, food, f| Range of amount of food to check for                                 | 0       |


## Examples
```yaml
  TargetConditions:
  - FoodLevel{a=<10} true
```
