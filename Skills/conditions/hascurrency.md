## Description
If the target has the given amount of vault currency


## Attributes

| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| amount    | a         | The amount of currency                                               |         |


## Examples
```yaml
  Conditions:
  - hascurrency{a=1000} true
```
