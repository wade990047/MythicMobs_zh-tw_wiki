## Description
Checks if the given [variable](/Skills/Variables)is set.


## Attributes

| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| variable  | name, n, var, key, k  | variable to match                                        |         |


## Examples

```yaml
  Conditions:
  - variableisset{var=target.dazed} true
```
