## Description
If the fall speed of the target is within the given range


## Attributes

| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| speed     | s         | The velocity to match                                                |         |


## Examples

```yaml
  Conditions:
  - fallspeed{s=>5} true
```


## Aliases
- [x] fallingspeed
