## 用途
召喚生物騎在目標身上，並踢下原騎乘者

## 細項設定
| 設定項 | 簡化寫法 | 用途 | 預設值 |
|-----------|-----------|----------------------------------------------------------------------|---------|
| passenger | p, t | 怪物類型   | |

  
## 範例
```yml
  Skills:
  - summonPassenger{type=MyZombie}
```

## 簡化寫法
- [x] passenger
- [x] summonRider
- [x] rider
