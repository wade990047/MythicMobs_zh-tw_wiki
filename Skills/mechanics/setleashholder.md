## Description
Sets the leash holder of the caster to the targeted entity


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| remove    | r, boolean, bool, b| If the lead should be removed instead of set                | false   |


## Examples
```yaml
  Skills:
  - setleashholder @NearestPlayer
```
