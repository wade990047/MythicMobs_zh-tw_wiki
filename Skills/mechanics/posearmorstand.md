Mechanic: PoseArmorStand
========================

Moves the extremities of an armorstand, the value is in radians.

細項設定
----------

| 設定項 | 簡化寫法 | 用途 | 預設值 |
|-----------|---------|-------------|---------------|
| head  | | | 0,0,0 |
| body  | | | 0,0,0 |
| leftarm   | | | 0,0,0 |
| rightarm  | | | 0,0,0 |
| leftleg   | | | 0,0,0 |
| rightleg  | | | 0,0,0 |

  

範例
--------

  Skills:
  - posearmorstand{head=45,0,0} @self ~onSpawn
  - ...
