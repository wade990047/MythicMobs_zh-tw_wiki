## Description
Sets the position of the dragon's podium (aka the point the dragon circles around) at the target location. paper 1.18.2+ only.

## Attributes
*This mechanic has no attributes*

## Example
```yaml
  Skills:
  - setDragonPodium @Location{c=125,50,-521}
```
