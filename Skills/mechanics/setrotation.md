用途
======================

設定目標的轉向(不適用玩家)

細項設定
----------

| 設定項 | 簡化寫法 | 用途 | 預設值 |
|-----------|---------|--------------------------------------------------|---------------|
| relative  | | 變化是否相對於目標 |   |
| yaw   | | 新的轉向 | 0 |
| pitch | | 新的角度   | 0 |

範例
--------
```yml
  Skills:
  - setrotation{relative=true;pitch=-45}
  - ...
```
