Mechanic: BossBorder
============================

Traps the player within a border around the caster.

Attributes
----------

| Attribute     | Aliases | Description                                                                   | Default Value |
|---------------|---------|-------------------------------------------------------------------------------|---------------|
| Range | r       | The border radius                                                   | 1             |

  

Examples
--------
```yml
Skills:
  - bossBorder{r=40}
```
