用途
----------------------

在指定位置變更控制桿狀態

**筆記:** 

控制桿會在經過設定的指定時間後恢復原狀 (預設值為 20 ticks / 1 秒).

細項設定
----------

| 設定項 | 簡化寫法 | 用途 | 預設值 |
|-----------|---------|-------------------------------------------------------------|---------------|
| duration  | d   | 控制桿的狀態要變更多久(單位:ticks) | 20|
| x | | X 座標 | 0 |
| y | | Y 座標 | 0 |
| z | | Z 座標 | 0 |

  

範例
--------
```yml
開啟隱藏門:
  Skills:
  - togglelever{d=600;x=15;y=67;z=-213}
```
