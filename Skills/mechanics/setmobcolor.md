Mechanic: Set Mob Color
=======================

Changes the target color, can only be applied to colorable mobs such as
shulkers or sheeps.

細項設定
----------

| 設定項 | 簡化寫法 | 用途 | 預設值 |
|-----------|---------|----------------|---------------|
| color | c   | The color name |   |

  

範例
--------

  Skills:
  - setcolor{color=blue} @self ~onSpawn
  - ...
