## Description
Targets all items in a radius around the caster


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| radius    | r         | The radius of the targeter                                           | 5       |


## Examples
```yaml
ExampleSkill:
  Skills:
  - pickupitem @ItemsInRadius{r=10}
```
