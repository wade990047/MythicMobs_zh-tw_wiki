## Description
Targets the nearest player in a radius

## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| radius    | r         | The radius of the targeter                                           | 5       |


## Examples
```yaml
ExampleSkill:
  Skills:
  - settarget @NearestPlayer{r=10}
```
