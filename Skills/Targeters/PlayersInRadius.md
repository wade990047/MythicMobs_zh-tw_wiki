## Description
Targets all players in the given radius around the caster


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| radius    | r         | The radius of the targeter                                           | 5       |


## Examples
```yaml
  Skills:
  - ignite @PIR{r=10}
```
